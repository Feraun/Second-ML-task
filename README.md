<h1>For start</h1>

1. Edit docker-compose, if necessary

  ``docker-compose up``

2. Create your own <i><b>.env</b></i> file

3. ``pip install -r requirements.txt``

4. Testing with Jupyter or FastApi (you can run tests in the <b>fastApi_tests</b> folder, after ``uvicorn main:app --reload``)
